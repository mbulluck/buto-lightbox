Buto Lightbox Video Embed jQuery plugin

https://github.com/ButoVideo/buto-lightbox

Brian's Buto

The Buto lightbox embed plugin has been created and tested with jQuery 1.7.2 to 1.9.1. It will search the page for any elements (usually divs) that you specify which have a Buto video id (eg 12345) . It will then talk to the Buto API and get the poster frames for that video. When you click on the poster frame you'll get a lightbox (courtesy of jackmoore) with the video in in. The script will detect the size of the div and resize the poster frame accordingly.

It's lightweight and fast, designed to be async and not block DOM render (in a much as is possible)

Usage

Ensure you have jquery 1.7x or above loaded first, it may work fine with earlier versions but is untested as such
