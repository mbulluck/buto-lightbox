Buto Lightbox Video Embed jQuery plugin

Buto is an enterprise online video platform

Brian's Buto

The Buto lightbox embed plugin has been created and tested with jQuery 1.9x. It will search the page for any elements (usually divs) that you specify which have a Buto video id (eg 12345) . It will then talk to the Buto API and get the poster frames for that video. When you click on the poster frame you'll get a lightbox (courtesy of jackmoore) with the video in in.

It's lightweight and fast, designed to be async and not block DOM render (in a much as is possible)

Usage

Ensure you have jquery 1.9x loaded first, it should work fine with earlier versions but is untested as such
